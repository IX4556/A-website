 思考笔记

一、到底什么是视觉传达？

  • 简单说：用视觉语言做信息传递、审美表达。把文字、图形、色彩、版式、影像，通过设计让信息好看、好懂、好传播。
  • 📐 平面设计
  • 🎨 品牌设计
  • 🖼️ 插画设计
  • 📱 UI 界面设计
  • 📦 包装设计
  • 🎬 广告与动态视觉

二、📚 基础必修课

  • 素描 / 色彩 / 速写、构成设计（平面 / 色彩 / 立体）
  • 版式设计、字体设计、图形创意、色彩原理
  • 摄影基础、艺术概论、设计史

三、💡 专业核心课

  • 平面设计、海报设计、书籍装帧、包装设计
  • 品牌 VI 设计、LOGO 设计
  • 广告创意、文案策划、插画设计、IP 形象设计
  • UI / 界面设计、交互基础、动态视觉 / 短视频剪辑

四、🛠️ 必学软件

  • 🎨 平面设计：PS、AI、InDesign
  • 🎬 动效制作：AE、PR
  • ✏️ 插画绘制：Procreate、SAI
  • 🚀 进阶工具：C4D（三维视觉）、Figma（UI）
